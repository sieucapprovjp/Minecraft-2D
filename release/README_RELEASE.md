# Minecraft 2D Release

## Cách chạy

Yêu cầu máy người chơi đã cài Java 17 hoặc mới hơn.

Mở terminal/cmd tại thư mục chứa file `.jar`, sau đó chạy:

```powershell
java -jar Minecraft-2D-1.0.0.jar
```

Nếu chạy bằng cách double-click không mở được game, hãy chạy bằng lệnh trên để xem thông báo lỗi.

## File trong bản release

- `Minecraft-2D-1.0.0.jar`: bản game desktop runnable jar.

## Ghi chú

Nếu muốn người chơi không cần cài Java, cần đóng gói thêm bản native/runtime installer thay vì chỉ phát hành file `.jar`.
